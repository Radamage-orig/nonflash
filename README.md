чтобы код работал вам нужно: 
1) Visual Studio 2022
2) "Консольное приложение (Майкрософт) C#"
3) версия .NET 8.0 (долгоспрочная поддержка)
4) Нажимайте сверху "Проэкт" и потом "Свойста (название вашего проэкта)
5) далее слева "Сборка" и "Общие"
6) потом меняете "Целевая платформа" на "x64"
7) после этого нажимайте "Проэкт" и "Управление пакетами NuGet..."
8) Нажимаете "Обзор" и пишете в поисковую строку "swed64"
9) выбираете самый верхний и устонавливауте
10) после установки во вкладку "Program.cs" вставляете весь код
11) потом нажимаете на кнопку сверху "Сборка" и "Собрать решение"
12) Готово!
